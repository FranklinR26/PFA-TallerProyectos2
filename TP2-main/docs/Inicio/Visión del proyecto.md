# **SPRINT 0: INICIO DEL PROYECTO**
## **Sistema Inteligente de Generación Óptima de Horarios Académicos**
## **Declaración de la Visión del Proyecto**
El proyecto tiene como visión desarrollar una **aplicación web inteligente, escalable y eficiente**, capaz de generar automáticamente horarios académicos óptimos en instituciones de educación superior que operan bajo un modelo de currículo flexible.

Se busca transformar el proceso tradicional de planificación académica, el cual suele ser manual, propenso a errores y poco eficiente, en un sistema automatizado que:

- Reduzca significativamente los conflictos de horarios
- Mejore la asignación de recursos académicos
- Facilite la toma de decisiones de coordinadores académicos
- Brinde a los estudiantes opciones de horarios más adecuadas a sus necesidades

A largo plazo, el sistema podrá adaptarse a diferentes contextos institucionales y escalar para soportar mayores volúmenes de información, integrándose con otros sistemas académicos existentes.
